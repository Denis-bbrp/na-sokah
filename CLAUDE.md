## Технологии

- Чистый HTML + CSS + vanilla JS (никаких фреймворков)
- Шрифты: Unbounded (заголовки десктоп), Golos Text (основной), Playfair Display (акценты мобилки)
- Всё в одном файле (inline CSS + JS)

## Секции сайта (обе версии)

1. **#hdr** — шапка (лого, Войти, Корзина)
2. **#hero** — главный баннер (оранжевый фон, 3 колонки с фото)
3. **#cats** — категории блюд (горизонтальный скролл)
4. **#sets** — Шеф-наборы (промо + список)
5. **#fav** — Хиты продаж (карточки)
6. **#mn** — Меню (блюда с вариантами порций)
7. **#coll** — Коллекции рецептов
8. **#testi** — Отзывы покупателей
9. **#ctr** — Счётчик
10. **#faq** — Частые вопросы (аккордеон)
11. **#ft** — Футер

### Модальные окна
- **#lov** — Логин (телефон + SMS)
- **#csb / #cov** — Корзина (сайдбар)
- **#docov** — Документы
- **#cookie** — Куки-баннер

## Отличия мобилки от десктопа

| Элемент | Десктоп | Мобилка |
|---|---|---|
| Шрифт заголовков | Unbounded | Playfair Display |
| Hero | сетка 3 колонки | горизонтальный скролл |
| Меню | grid 2 колонки | список (карточки горизонтальные) |
| Коллекции | grid 2 колонки | flex column |
| Отзывы | grid 2 колонки | горизонтальный скролл |
| Навигация | нет | bottom nav (#bnav) |
| Логин | модал по центру | bottom sheet снизу |
| Корзина | сайдбар 420px | полный экран |
