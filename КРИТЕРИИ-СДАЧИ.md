# Критерии топового сайта и план сдачи НА СОКАХ.Еда

---

## Часть 1 — Критерии топового сайта

### Дизайн и UX
- Единая дизайн-система (цвета, типографика, отступы — без разнобоя)
- Читаемость: контраст текста достаточный, текст не сливается с фоном
- Все кнопки и ссылки имеют hover/active состояния
- Нет "мёртвых" кнопок (каждый элемент что-то делает)
- Плавные анимации (не раздражают, не тормозят)
- Логика навигации понятна без объяснений

### Адаптивность
- Корректно работает на: 320px, 375px, 414px, 768px, 1024px, 1280px, 1440px+
- Нет горизонтальной прокрутки на мобилке
- Тапабельные зоны не меньше 44px (стандарт iOS/Android)
- Шрифты не мельче 13px на мобилке

### Скорость
- Время загрузки < 3 секунды на мобиле
- Изображения сжаты (WebP вместо PNG/JPG где возможно)
- Нет блокирующих скриптов в <head>

### SEO
- <title> уникальный, до 60 символов
- <meta name="description"> до 160 символов
- Open Graph теги (og:title, og:description, og:image) — для красивого превью при шаринге в соцсетях
- Favicon — иконка во вкладке браузера
- Семантическая разметка HTML (заголовки h1→h2→h3 по иерархии)

### Контент
- Весь текст реальный (не Lorem Ipsum)
- Все ссылки ведут куда надо (нет href="#")
- Реальные контакты: телефон, email, адрес
- Реальные ссылки на соцсети и мессенджеры
- Юридические документы: оферта + политика конфиденциальности

### Аналитика
- Яндекс.Метрика установлена (обязательно для рунета)
- Настроены цели: "Добавить в корзину", "Оформить заказ", "Позвонить"

### Технические требования
- Сайт работает по HTTPS
- Нет ошибок в консоли браузера (F12 → Console)
- Все внешние ресурсы загружаются (нет битых ссылок на картинки/шрифты)

## Часть 3 — План доработок по приоритету

### 🔴 Приоритет 1 — Обязательно перед сдачей

1. **SEO-теги** — добавить в <head> обоих файлов:
   - `<meta name="description">`
   - `<meta property="og:title">`, `og:description`, `og:image`

2. **Favicon** — иконка во вкладке браузера:
   - favicon.ico (32×32 пикселя)
   - apple-touch-icon.png (180×180 для iOS)

3. **Реальные ссылки** — заменить заглушки:
   - Соцсети: Telegram, VK, WhatsApp
   - Телефон: `href="tel:+78007001520"`
   - Email: `href="mailto:privacy@nasokah.ru"`

4. **Яндекс.Метрика** — счётчик в оба файла (na-sokah.html и mobile.html)

5. **Оформить заказ** — минимум отправка данных на email или в Telegram-бот

### 🟡 Приоритет 2 — Желательно

6. Оптимизация фото (PNG → WebP, меньший размер файлов)
7. Страница 404 (когда пользователь попал не туда)
8. Проверка на всех экранах через DevTools

### 🟢 Приоритет 3 — После запуска

9. Реальная SMS-авторизация (сервис SMSC.ru или СМС.ру)
10. История заказов пользователя
11. Подключение к CRM или Google Таблицам

## Часть 4 — Финальный чек-лист перед сдачей

### Визуальная проверка
- [ ] Открыть на iPhone (375px) — нет горизонтального скролла
- [ ] Открыть на Android (360px) — шрифты читаемы
- [ ] Открыть на планшете (768px) — сетки не ломаются
- [ ] Открыть на десктопе (1440px) — всё пропорционально

### Функциональная проверка
- [ ] Добавить товар в корзину → сумма обновилась
- [ ] Удалить товар из корзины → корзина пересчиталась
- [ ] Фильтр категорий — блюда фильтруются корректно
- [ ] FAQ — каждый вопрос открывается и закрывается
- [ ] Форма входа — номер телефона принимается
- [ ] Документы (оферта, политика) — открываются в модальном окне
- [ ] Cookie-баннер — закрывается и не появляется снова

### Техническая проверка
- [ ] F12 → Console → нет красных ошибок
- [ ] F12 → Network → нет 404 запросов (битых ссылок)
- [ ] Все внешние ссылки ведут куда надо
- [ ] Шрифты загружаются (Google Fonts работают)
- [ ] PageSpeed Insights → мобилка > 70 баллов

### SEO
- [ ] <title> есть и уникальный
- [ ] <meta description> есть
- [ ] Favicon отображается во вкладке
- [ ] При шаринге ссылки в Telegram — показывается красивое превью с картинкой

### Аналитика
- [ ] Яндекс.Метрика установлена в оба файла
- [ ] В Метрике видны визиты после тестовых переходов
